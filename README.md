# Personal JARVIS 🤖

Personal JARVIS is a low-cost AI-powered assistant robot designed for students. It can understand voice commands, answer questions using AI, and perform basic robotic movements.

## 🚀 Features
- 🎤 Voice Control (Hindi + English)
- 🤖 AI-based Question Answering
- 🚗 Robot Movement (Forward, Back, Left, Right, Stop)
- 🔊 Speaker Output
- 📡 Bluetooth Communication
- 💡 Low Cost Solution

## 🛠️ Tech Stack
- Arduino UNO
- Python (AI + Voice Processing)
- Bluetooth Module (HC-05)
- Motor Driver (L298N)

## 📌 How It Works
1. User gives voice input
2. Voice converts to text
3. Command is processed
4. If movement → Robot moves
5. If question → AI answers
